# Evaluation Reports

Current canonical report: [`CANONICAL-VALIDATION-REPORT.md`](CANONICAL-VALIDATION-REPORT.md).

Older evaluation run artifacts in this folder are historical snapshots unless they are explicitly regenerated by `source.scripts.run_product_validation`. This includes dated run reports, goldset answer/judgement reports, contract-reconciliation runs, linking smokes, and symbol-query smokes. Use the canonical report for current product-validation status; older files are retained only for audit history and comparison.

Regenerate the canonical report with:

```bash
python -m source.scripts.run_product_validation
```
